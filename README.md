# 📧 AI Email Assistant

An intelligent email analyzer built with Python and NLP.

## 🚀 Features
- Detect email importance
- Summarize emails
- Generate suggested replies
- Simple web UI with Streamlit
- Local ML models (no API needed)

## 🛠 Tech Stack
- Python
- Transformers (HuggingFace)
- Streamlit
- SQLite

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
